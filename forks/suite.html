This is a temporary folder. It shall be disposed after it has served its purpose. I have no idea, how long this will take.

Please let me know in case someone dislikes beeing in this compilation.

Compared to the original files, the present files are processed as follows :

  • Convert from ANSI to UTF-8-with-BOM
  • Replace tabs by triple spaces
  • Remove trailing spaces
  • Convert line endings from \n to \r\n, just for my conveniece. Sorry, if you'd like it as it was. I am ready to negotiate.
  • Adjust page titles
  • Connect to project stylesheet
  • Supplement shortcut icon links
  • Supplement navigation headers
  • Sedate the beep.mp2 dead links
  • Supplement page IDs
  • Validate with Tidy. Note: This went surprisingly flawless — No errors in all that pages! Only one missing alt attribute and two curious "Warning: '<' + '/' + letter not allowed here". I remember this warning from other occasions. You can see a method how to get rid of it in page tools.html at issue 20160612°0251 'Letter quirk with JavaScript inside HTML'.
  • In nearly all JavaScript code, I supplemented semicoli at command ends.
  • Without any ado, I changed == to === and != to !==.

After those sweeping modifications, the code is partially modified. Those modifications shall be marked with comments, typically with quadruple slashes (////).
