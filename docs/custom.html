PurpleTerms Custom Demo

This page is an example of integrating PurpleTerms into a custom environment. And it demonstrates AJAX communicaton, or XHR respectively, with a PHP backend module. The present AJAX/XHR feature does not (yet) utilize the PurpleTerms built-in connect() function, but instead an external custom module custom.dafterm.js.

The box knows two commands. beep should always work. spin works only with PHP available. PHP must be set to recognize PHP code inside HTML, e.g. like in your local development environment. The spinner has no stop command, it will stop after erroneous input or page refresh.

Currently exists no online version of this page with PHP available.
