PurpleTerms Custom Demo

This pae is called 'Custom Demo', because there are more files involved, than only terminal.js. Those provide you boiler plate code how to integrate PurpleTerms into your system, especially how to communicate via AJAX to a PHP backend.

The AJAX functionality will only be available, if you run this page with PHP available, e.g. from your localhost development server.
