Custom Demo

This page is an example of integrating Terminal.js into a custom environment. And it demonstrates submitting AJAX requests, or XHR respectively, to a backend PHP module. The AJAX/XHR feature here does not (yet) utilize the Terminal.js built-in connect() function, but a custom module custom.dafterm.js.

The beep command should always work. The spin command works only with PHP available. PHP must be set to recognize PHP code inside HTML, e.g. in your local development environment. The spinner has no stop command, it will stop after erroneous input or page refresh.

Currently exists no online version of this page with PHP available.
