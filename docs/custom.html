Purple Terminals Custom Demo

This page is called Custom Demo, because beside purpleterms.js itself, custom files are involved. This provides you boiler plate code to communicate from PurpleTerms via AJAX with a PHP backend module.

The demo runs fully functional only if PHP is available, e.g. on your localhost development server.
